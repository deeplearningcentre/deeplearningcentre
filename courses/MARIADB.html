MariaDB Learning Lecture Plan by Deep Learning Centre Gurgaon

This lecture plan will guide you through various topics related to MariaDB. MariaDB is an open-source relational database management system, a drop-in replacement for MySQL.

Lecture Plan Overview

Week Topic Description
Week 1 Introduction to MariaDB Understanding relational databases, introduction to MariaDB, and installing MariaDB.
Week 2 Basic SQL Commands Learning basic SQL queries: SELECT, INSERT, UPDATE, DELETE.
Week 3 Database Design & Normalization Designing efficient databases, understanding normalization, and creating tables with constraints.
Week 4 Advanced SQL Queries Working with joins, subqueries, and complex SQL queries.
Week 5 Indexes & Performance Optimization Understanding how indexes work, and learning how to optimize SQL queries and database performance.
Week 6 Stored Procedures & Triggers Creating stored procedures, triggers, and user-defined functions to automate tasks in MariaDB.
Week 7 Database Backup & Recovery Learning the importance of backups, methods of backup, and restoring data from backups.
Week 8 Security & User Management Managing users and permissions, securing databases, and best practices for database security.
Week 9 Replication & High Availability Configuring MariaDB replication for high availability and scalability.
Week 10 MariaDB Administration & Monitoring Using MariaDB tools to administer, monitor, and troubleshoot MariaDB servers.

Assignments & Projects

Throughout this course, you will work on the following assignments and projects:

  • Project 1: Design and implement a basic database for a library system.
  • Project 2: Optimize a set of queries for a large dataset.
  • Project 3: Set up replication between two MariaDB servers and test data consistency.
  • Project 4: Implement a backup and recovery plan for a database.
