HashiCorp Vault Lecture Plan by Deep Learning Centre Gurgaon

Lecture Overview

This lecture series provides a comprehensive introduction to HashiCorp Vault, a tool for securely storing and accessing secrets in modern infrastructure. The plan is divided into two days, focusing on theory, practical demonstrations, and hands-on exercises.

Day 1: Introduction & Core Concepts

  • Introduction to HashiCorp Vault
    • What is Vault?
    • Use cases for Vault in infrastructure security
    • Overview of key features: Secrets Management, Encryption as a Service, Access Control
  • Vault Core Concepts
    • Secrets: What are they, and why do we need to protect them?
    • Authentication Methods (e.g., AppRole, LDAP, AWS IAM)
    • Policies and Access Control
  • Vault Installation & Setup
    • Installing Vault on different platforms (Linux, Mac, Windows)
    • Starting Vault in Development Mode
    • Initializing and Unsealing Vault
  • Hands-On Session: Day 1
    • Setup Vault in development mode
    • Store and retrieve a secret via Vault CLI
    • Understand basic commands like `vault status`, `vault secrets enable`, etc.

Day 2: Advanced Topics & Practical Applications

  • Vault Architecture & Advanced Features
    • Vault storage backends (e.g., Consul, File, etc.)
    • Dynamic Secrets (e.g., Database credentials)
    • Encryption as a Service
  • Managing Secrets with Vault
    • Creating, reading, updating, and deleting secrets
    • Accessing secrets using the API, CLI, and UI
    • Creating Vault policies and roles
  • Security Best Practices
    • Securing Vault with TLS
    • Policy Management and Implementing Least Privilege
    • Disaster Recovery: Backup and restore strategies
  • Hands-On Session: Day 2
    • Setup a production-like Vault configuration
    • Create and manage policies, roles, and dynamic secrets
    • Implement TLS security on Vault server
