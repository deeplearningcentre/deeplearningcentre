Linux Learning Lecture Plan by Deep Learning Centre Gurgaon

Lecture 1: Introduction to Linux

Objective:

Introduce students to the Linux operating system, its history, and basic concepts.

Topics:

  • Overview of Operating Systems
  • Why Linux?
  • History and evolution of Linux (Linus Torvalds, 1991)
  • Key components: Kernel, Shell, File System
  • Types of Linux Distributions: Ubuntu, Fedora, CentOS, Debian, Arch Linux
  • Basic Terminology: Kernel, Shell, Terminal, Desktop Environment
  • File System Hierarchy: Root `/`, `/home`, `/etc`, etc.
Hands-On Activity:
  • Boot into a live Linux environment (using VirtualBox, VMware, or WSL for Windows users)
  • Explore the graphical user interface (GUI)

Lecture 2: Linux Command Line Basics

Objective:

Introduce the Linux command line interface (CLI) and common commands.

Topics:

  • Command Line Interface vs. Graphical User Interface
  • Navigating the File System: `pwd`, `cd`, `ls`, `mkdir`, `rmdir`, `touch`
  • File Management: `cp`, `mv`, `rm`, `cat`, `less`, `more`, `file`
  • Understanding file permissions: `chmod`, `chown`, `chgrp`
Hands-On Activity:
  • Basic navigation and file manipulation through terminal

Lecture 3: Advanced Command Line and Text Processing

Objective:

Explore more advanced command-line tools and text-processing utilities.

Topics:

  • File Searching and Manipulation: `find`, `locate`, `grep`, `which`
  • Text Editors: Introduction to `nano`, `vim`, and `emacs`
  • Text Processing: `cat`, `head`, `tail`, `cut`, `sort`, `uniq`, `wc`
  • Piping commands (`|`) and redirecting output (`>`)
Hands-On Activity:
  • Using `grep` and `find` for searching files and text processing tools in real scenarios

Lecture 4: Package Management in Linux

Objective:

Teach how to manage software packages on different Linux distributions.

Topics:

  • Understanding Package Management
  • Overview of package managers: APT, YUM/DNF, Pacman
  • Installing, Removing, and Updating Packages: `apt-get`, `apt`, `yum`, `dnf`, `pacman`
Hands-On Activity:
  • Install and update a package using the command line

Lecture 5: Linux System Administration Basics

Objective:

Introduce basic system administration tasks like user management and system monitoring.

Topics:

  • User and Group Management: `useradd`, `usermod`, `groupadd`, `passwd`, `chage`
  • System Monitoring and Performance: `top`, `htop`, `free`, `df`, `du`, `ps`
  • Process Management: `kill`, `pkill`, `killall`
Hands-On Activity:
  • Add a user and monitor system performance

Lecture 6: Linux Networking and Security

Objective:

Introduce networking concepts in Linux and basic security measures.

Topics:

  • Linux Networking: `ifconfig`, `ip`, `ping`, `netstat`, `ss`
  • Firewall and Security Tools: Introduction to `iptables` and `ufw`
  • SSH and Remote Access: `ssh`
Hands-On Activity:
  • Configuring basic firewall rules and using `ssh` to connect to another system

Lecture 7: Scripting and Automation in Linux

Objective:

Teach basic shell scripting to automate tasks.

Topics:

  • Introduction to Shell Scripting: Creating and running a script
  • Writing Basic Shell Scripts: Variables, Conditionals, and Loops
  • Automation Tools: `cron` for scheduling tasks
Hands-On Activity:
  • Write a basic shell script to automate a task and schedule it with `cron`

Lecture 8: Troubleshooting and Linux System Recovery

Objective:

Teach basic troubleshooting techniques and system recovery.

Topics:

  • Basic Troubleshooting: Common Linux problems and solutions
  • System Recovery: Booting into rescue mode and repairing the bootloader (GRUB)
Hands-On Activity:
  • Troubleshooting a common issue (e.g., network problem, file permission problem)

Lecture 9: Advanced Linux Topics (Optional)

Objective:

Introduce advanced topics for students who want to deepen their Linux knowledge.

Topics:

  • Linux Kernel Compilation: Basic concepts of kernel and modules
  • Virtualization in Linux: Docker, LXC, VirtualBox
  • Linux in the Cloud: Introduction to cloud platforms like AWS, Azure, Google Cloud

Wrap-Up and Final Q&A

  • Review of key concepts covered
  • Final project or assignment (e.g., setting up a web server, writing a bash script)
  • Q&A session
