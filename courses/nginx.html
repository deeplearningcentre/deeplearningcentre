NGINX Network & Security Learning Lecture Plan by Deep Learning Centre Gurgaon

Welcome to the NGINX learning series. This lecture plan will guide you through various aspects of using NGINX for network and security management. Each section is divided into individual lectures with practical steps and explanations.

Lecture 1: Introduction to NGINX

Duration: 1 Hour

  • Overview of NGINX as a web server, reverse proxy, and load balancer
  • Architecture of NGINX
  • Installation and Configuration
  • Basic commands for managing NGINX

Lecture 2: Basic NGINX Networking

Duration: 1 Hour 30 Minutes

  • Configuring server blocks
  • Understanding HTTP and HTTPS in NGINX
  • Setting up SSL/TLS
  • Managing different site configurations (virtual hosts)

Lecture 3: Load Balancing with NGINX

Duration: 1 Hour 30 Minutes

  • Overview of load balancing strategies (round-robin, IP-hash, etc.)
  • Configuring NGINX as a load balancer
  • Health checks for backend servers
  • Scaling applications with NGINX

Lecture 4: NGINX Security Essentials

Duration: 2 Hours

  • Setting up basic HTTP authentication
  • Using SSL/TLS for secure communication
  • Configuring NGINX for secure cookies and headers
  • Blocking malicious requests and rate limiting
  • Secure server block configurations

Lecture 5: Advanced Security Configurations in NGINX

Duration: 2 Hours

  • Using NGINX to prevent DDoS attacks
  • Setting up Web Application Firewall (WAF) with NGINX
  • Configuring HTTPS with HTTP Strict Transport Security (HSTS)
  • Advanced rate limiting and IP blocking techniques

Lecture 6: Performance Optimization and Security Auditing

Duration: 1 Hour 30 Minutes

  • Optimizing NGINX for high performance
  • Understanding and configuring caching in NGINX
  • Log analysis for security auditing
  • NGINX security best practices and updates
