Operating Systems - Lecture Plan

Course Duration: 12 Weeks

Week 1: Introduction to Operating Systems

Objectives:

  • Understand the role and functions of an Operating System.
  • Differentiating between types of operating systems.
  • Discussing the historical development of Operating Systems.

Topics:

  • Definition and Purpose of an Operating System
  • Types of Operating Systems (Batch, Time-Sharing, Real-Time, Distributed)
  • History of Operating Systems
  • Overview of Modern Operating Systems (Linux, Windows, macOS)

Activities:

  • Lecture and discussion
  • Brief historical timeline of OS development
  • Comparison of popular Operating Systems

Week 2: System Architecture and OS Components

Objectives:

  • Understand system architecture concepts.
  • Learn about the major components and layers of an OS.

Topics:

  • OS Structure: Monolithic vs Microkernel Architecture
  • OS Components: Kernel, Shell, File System, Device Drivers
  • System Calls and APIs
  • Virtual Machines and Emulation

Activities:

  • Diagrams of OS architecture
  • Explanation of system calls with examples
  • Hands-on with basic terminal commands

Week 12: Case Studies and Modern OS

Objectives:

  • Explore real-world applications and use cases of modern OS.

Topics:

  • Case Study: Linux OS Architecture and Features
  • Case Study: Windows OS Architecture
  • Mobile Operating Systems: Android, iOS
  • Future Trends in Operating Systems (Cloud, IoT, AI Integration)

Activities:

  • Discussion and comparison of Linux vs Windows vs macOS
  • Final project or presentation on a modern OS feature
