Ruby on Rails Web Development Learning Plan by Deep Learning Centre Gurgaon

Welcome to the Ruby on Rails learning journey! This plan is designed to help you build a solid foundation in web development using the Ruby on Rails framework. Below is a week-by-week breakdown of the topics you will cover.

Week 1: Introduction to Ruby on Rails

  • What is Ruby on Rails? Overview of the MVC framework.
  • Setting up Ruby on Rails environment.
  • Basic Ruby programming concepts (variables, data types, loops, and conditionals).
  • Introduction to the Rails command-line interface (CLI).
  • Creating your first Rails application.

Week 2: Understanding MVC Architecture

  • Introduction to MVC (Model-View-Controller) pattern.
  • Creating models and migrations.
  • Generating controllers and views.
  • How routing works in Rails.
  • Building a simple CRUD (Create, Read, Update, Delete) application.

Week 3: Active Record & Database Interaction

  • Introduction to Active Record.
  • Understanding database migrations and schema.
  • How to use Active Record queries for fetching data.
  • Associations between models (has_many, belongs_to, has_one).
  • Validations and callbacks in Active Record models.

Week 4: Working with Views and Templates

  • Understanding ERB (Embedded Ruby) templates.
  • Displaying data in views.
  • Using partials and layouts.
  • Introduction to helpers for simplifying view code.
  • Working with forms in Rails.

Week 5: Introduction to Rails RESTful Routes

  • Understanding REST and how it's implemented in Rails.
  • Defining RESTful routes in Rails.
  • Working with the resourceful routing system.
  • Building a RESTful API using Rails.

Week 6: Authentication & Authorization

  • Introduction to user authentication in Rails.
  • Using Devise gem for authentication.
  • Authorization with CanCanCan gem.
  • Building user registration and login functionality.
  • Securing routes based on user roles.

Week 7: Rails Testing

  • Introduction to testing in Rails with RSpec.
  • Unit tests for models and controllers.
  • Integration tests for simulating real user interactions.
  • Test-driven development (TDD) in Rails.
  • Writing feature tests using Capybara.

Week 8: Deploying Rails Application

  • Understanding deployment environments.
  • Introduction to Heroku for Rails deployment.
  • Setting up a production database (PostgreSQL).
  • Deploying your first Rails app to Heroku.
  • Understanding production and development configurations.
