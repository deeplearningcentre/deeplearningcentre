Learning TypeScript in HTML

This lecture plan covers the basics of using TypeScript with HTML and building web applications.

Week 1: Introduction to TypeScript & HTML Setup

  • Introduction to TypeScript as a typed superset of JavaScript.
  • Differences between TypeScript and JavaScript.
  • Key features: static typing, interfaces, classes.
  • Benefits of using TypeScript: early error detection, maintainability, etc.

Week 2: Core TypeScript Features for Web Development

Lecture 3: Basic TypeScript Syntax

  • Declaring variables with `let`, `const`, and `var`.
  • Using types like `string`, `number`, `boolean` in TypeScript.
  • Type inference and type annotation.
  • Function declarations and return types.
let message: string = "Hello, TypeScript!";

Lecture 4: Using TypeScript with the DOM

  • Access HTML elements using TypeScript.
  • Adding event listeners and interacting with user input.
  • TypeScript and DOM manipulation (querySelector, addEventListener).

let button = document.querySelector("#myButton")!;
button.addEventListener("click", () => {
    let message = document.querySelector("#message");
    message!.innerText = "Button clicked!";
});

Week 3: Advanced TypeScript Concepts

Lecture 5: TypeScript Classes and Interfaces

  • Creating classes in TypeScript and using constructors.
  • Interfaces for type-checking objects and classes.

interface Person {
    name: string;
    age: number;
}

class User implements Person {
    constructor(public name: string, public age: number) {}
    greet() {
        return `Hello, my name is ${this.name} and I am ${this.age} years old.`;
    }
}

let user = new User("Alice", 25);
console.log(user.greet());

Lecture 6: TypeScript and Complex Forms

  • Working with HTML forms using TypeScript.
  • Handling form validation and submission.

document.querySelector("form")!.addEventListener("submit", (event) => {
    event.preventDefault();
    let username = (document.querySelector("#username") as HTMLInputElement).value;
    if (username.trim() === "") {
        alert("Username cannot be empty.");
    } else {
        alert("Form submitted successfully!");
    }
});

Week 4: Building a Simple Web Application with TypeScript and HTML

Lecture 7: Building a To-Do List Application

  • Create a basic To-Do list application.
  • Use TypeScript to add, remove, and store tasks.
  • Understand TypeScript's interaction with localStorage.

let taskInput = document.querySelector("#taskInput") as HTMLInputElement;
let addButton = document.querySelector("#addButton") as HTMLButtonElement;
let taskList = document.querySelector("#taskList")!;

addButton.addEventListener("click", () => {
    if (taskInput.value.trim() !== "") {
        let listItem = document.createElement("li");
        listItem.textContent = taskInput.value;
        taskList.appendChild(listItem);
        taskInput.value = "";
    }
});
