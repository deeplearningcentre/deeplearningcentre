VMware Learning Plan by Deep Learning Centre Gurgaon

Week 1: Introduction to VMware and Virtualization

Objective:

Learn the fundamentals of VMware, virtualization concepts, and the advantages of using VMware for virtualization.

Topics Covered:

  • What is VMware?
  • Types of Virtualization: Full Virtualization vs Paravirtualization
  • How VMware works: ESXi, vCenter Server, and vSphere
  • Installing VMware Workstation/ESXi/VMware Player

Assignments:

  • Install VMware Workstation or VMware Player on your machine.
  • Create your first virtual machine (VM) in VMware Workstation.

Week 2: VMware ESXi and vCenter Server

Objective:

Learn about VMware ESXi, the hypervisor, and vCenter Server for managing VMware environments.

Topics Covered:

  • Introduction to VMware ESXi and its role in virtualization
  • Installing VMware ESXi
  • Overview of vCenter Server and its importance
  • Connecting to vCenter Server for VM management

Assignments:

  • Install VMware ESXi on a physical or virtual machine.
  • Configure vCenter Server and connect to ESXi for VM management.

Week 3: Managing Virtual Machines in VMware

Objective:

Learn how to manage virtual machines in a VMware environment, including creation, configuration, and lifecycle management.

Topics Covered:

  • Creating and Configuring Virtual Machines
  • VM Hardware Configuration: CPU, Memory, Storage, Network Adapters
  • Snapshot and Cloning of Virtual Machines
  • Managing VM Lifecycle: Powering on, off, and restarting VMs

Assignments:

  • Create a virtual machine and configure the hardware (CPU, RAM, Disk, etc.).
  • Take a snapshot of the VM and perform a clone operation.

Week 4: VMware Networking

Objective:

Learn about VMware networking options for virtual machines and how to configure networking for VMs.

Topics Covered:

  • Understanding VMware Networking Concepts
  • Types of Network Adapters in VMware (Bridged, NAT, Host-Only)
  • Creating and Managing Virtual Switches (vSwitch)
  • Configuring Network Security Settings and VLANs in VMware

Assignments:

  • Configure different types of network adapters for virtual machines.
  • Set up a virtual switch (vSwitch) and test connectivity between VMs.

Week 5: VMware Storage Management

Objective:

Learn how to configure and manage storage for virtual machines in VMware environments.

Topics Covered:

  • Understanding VMware Storage Concepts
  • Types of Storage in VMware: VMFS, NFS, and iSCSI
  • Adding and Managing Virtual Disks for VMs
  • Using Datastores for VM Storage

Assignments:

  • Add and configure a virtual disk for a virtual machine.
  • Set up a datastore for storing virtual machine files and data.

Week 6: VMware vSphere and High Availability

Objective:

Understand VMware vSphere and how to set up high availability and fault tolerance for virtualized environments.

Topics Covered:

  • Introduction to VMware vSphere
  • Setting up VMware HA (High Availability) for VMs
  • Understanding VMware Fault Tolerance and its configuration
  • vMotion: Live Migration of VMs

Assignments:

  • Enable VMware HA and test failover with virtual machines.
  • Use vMotion to migrate a virtual machine between hosts.

Week 7: VMware Resource Management and Monitoring

Objective:

Learn how to monitor and manage resources such as CPU, memory, and storage in a VMware environment.

Topics Covered:

  • Understanding VMware Resource Allocation and Limits
  • Monitoring CPU, Memory, and Disk usage in VMware vCenter
  • Using VMware Distributed Resource Scheduler (DRS)
  • Resource Pools in VMware

Assignments:

  • Set resource allocation limits for a virtual machine.
  • Configure a resource pool and observe resource distribution.

Week 8: VMware Security and Best Practices

Objective:

Learn how to secure VMware environments and follow best practices for system administration.

Topics Covered:

  • Securing VMware ESXi Hosts and vCenter Server
  • Role-Based Access Control (RBAC) in VMware
  • Configuring Firewalls and SSL/TLS for VMware
  • Backup and Restore of VMware Virtual Machines

Assignments:

  • Configure firewall settings for VMware ESXi and vCenter Server.
  • Perform a backup and restore of a virtual machine in VMware.

Week 9: VMware Automation with PowerCLI

Objective:

Learn how to automate VMware tasks using PowerCLI.

Topics Covered:

  • Introduction to PowerCLI and its usage
  • Basic PowerCLI commands for VMware management
  • Automating common tasks like VM creation, deletion, and snapshot management

Assignments:

  • Install VMware PowerCLI and run basic commands to manage your VMware environment.
  • Automate the creation and deletion of virtual machines using PowerCLI scripts.

Week 10: VMware Troubleshooting and Maintenance

Objective:

Learn VMware troubleshooting and maintenance techniques to ensure a healthy virtualized environment.

Topics Covered:

  • Common VMware issues and how to troubleshoot them
  • Log files in VMware ESXi and vCenter Server
  • Upgrading VMware ESXi and vCenter Server

Assignments:

  • Troubleshoot a common VMware issue (e.g., VM not starting, resource issues).
  • Upgrade VMware ESXi and vCenter Server to the latest version.
