Wireshark Network & Security Learning Lecture Plan by Deep Learning Centre Gurgaon

Course Overview

This lecture plan focuses on learning how to use Wireshark, a network protocol analyzer, for network troubleshooting, traffic analysis, and security monitoring.

Lecture 1: Introduction to Wireshark

  • Overview of network protocols
  • Introduction to Wireshark and its uses
  • Installing and setting up Wireshark
  • Understanding the Wireshark interface
  • Basic network traffic capture

Lecture 2: Capturing and Analyzing Traffic

  • Configuring capture filters
  • Understanding packet details
  • Identifying common protocols (e.g., HTTP, TCP, UDP, DNS)
  • Wireshark display filters and searching packets
  • Using the "Statistics" menu to analyze traffic

Lecture 3: Advanced Filtering and Analysis Techniques

  • Advanced display filters (e.g., IP addresses, ports, protocols)
  • Following a TCP stream
  • Reassembling fragmented packets
  • Coloring rules for packet analysis
  • Exporting packet captures for further analysis

Lecture 4: Security Analysis with Wireshark

  • Detecting common network attacks (e.g., DDoS, ARP poisoning)
  • Identifying unencrypted sensitive data
  • Analyzing SSL/TLS traffic
  • Wireshark as a forensic tool
  • Setting up alerts for suspicious activity

Lecture 5: Wireshark in Real-World Scenarios

  • Network troubleshooting with Wireshark
  • Wireshark in corporate security audits
  • Using Wireshark for penetration testing
  • Wireshark in incident response and malware analysis

Lecture 6: Hands-On Lab and Review

  • Lab 1: Capturing and analyzing traffic on a local network
  • Lab 2: Identifying security issues in a sample capture file
  • Review of key concepts and techniques learned
  • Q&A session
